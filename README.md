# 📔 MemoryVault - Your Personal Digital Diary

## 📺 Link: https://emman-21.github.io/MemoryVault/

# ✨ Features

### 📝 **Core Features**
- ✏️ **Write & Edit** - Create beautiful diary entries
- 🗑️ **Delete Entries** - Remove with confirmation dialog
- 🔒 **100% Private** - Everything stored locally in your browser
- 💾 **Auto-save** - Never lose your thoughts

### 🎨 **Design & Experience**
- 🏛️ **Vintage Leather Theme** - Classic diary aesthetic
- 📄 **Ruled Paper Effect** - Authentic writing feel
- 📱 **Fully Responsive** - Works on all devices
- 🎭 **Mood Tracking** - Tag entries with emotions (😊 😐 😔 🤩 🥰)

### 🗂️ **Organization**
- 🗓️ **Smart Filters** - View by Today/Week/Month/All
- 📊 **Writing Stats** - Track entries, streak, words
- 🔍 **Quick Access** - Recent entries sidebar
- 📤 **Export Data** - Backup as JSON file

# 🚀 **Quick Start**

1. 📥 **Download** all three files:
   - `index.html`
   - `memoryvault.css` 
   - `memoryvault.js`

2. 📁 **Put them** in the same folder

3. 🌐 **Open** `index.html` in any browser

4. ✍️ **Start writing!**

# 🎯 **How to Use**

### ✨ **Creating Entries**
- ➕ Click **"New Entry"** button
- 📝 Write your thoughts
- 😊 Pick a mood
- 💾 **Save** or let auto-save work

### 🗑️ **Managing Entries**
- ✏️ **Click** entry in sidebar to edit
- ❌ **Hover & click X** to delete specific entry
- 🗑️ Use **Delete button** for current entry
- 📅 **Filter** by time period

### ⚡ **Shortcuts**
- `Ctrl/Cmd + D` - Delete entry
- `Delete key` - Delete selected entry
- ⏱️ Auto-saves every 30 seconds
